# Robot_Ksu_Bot
🤖 Telegram бот для анализа PDF документов о роботизации с использованием GigaChat AI.

## 🌟 Особенности
- Автоматическое определение языка вопроса
- Ответ на том же языке, что и вопрос
- Поддержка разных языков (RU, EN)
- Специализация на роботах и их применении
- Анализ множества PDF документов и Mackdown-файлов
